# GD01 project
- Digital clock powered by a Raspberry Pico 2W
- Time & date synchronization with NTP server
- 7-segment displays managed by shift register SN74HC595N
- High-current Darlington transistor array ULN2803A
- Wireless control via an HTML interface or 3x buttons
- Microdot server and Micropython script under MIT license

# GD02 project
- DIY through-hole LED board powered by a Raspberry Pico 2W
- 16 yellow LEDs driven by 2× TPIC power shift registers
- All LEDs on (mode 1) or Minute indicator with animations (mode 2)
- Time synchronization with NTP server
- Wireless control via an HTML interface or 2x buttons
- Microdot server and Micropython script under MIT license
